# 查询指南模块设计

**版本**: 1.0
**日期**: 2026-03-17
**状态**: 已确认

---

## 1. 背景

当前知识库模块存储的是用户上传的业务文档，但实际使用中发现两类内容用途不同：

| 内容类型 | 用途 | 建议归属 |
|----------|------|----------|
| 政策法规、企业文化等 | 通用知识，不一定与具体数据源相关 | 知识库（保持现状） |
| 表/字段说明、SQL参考、业务规则、注意事项 | 辅助查询，与数据源强绑定 | 查询指南（新增） |

本次新增"查询指南"模块，专门用于存储与数据源绑定的查询辅助文档。

---

## 2. 设计目标

1. **与数据源绑定**：每个数据源有独立的查询指南
2. **混合输入**：支持上传文档 + 手动备注
3. **智能体可探索**：通过 grep/cat 等工具自由查阅
4. **简洁易用**：侧边抽屉形式编辑，不增加页面复杂度

---

## 3. 技术设计

### 3.1 存储结构

**文件系统**：
```
data/query_guides/
└── {datasource_id}/
    ├── uploaded/              # 用户上传的文档
    │   ├── 数据字典.md
    │   └── 业务规则.docx
    └── notes.md               # 手动备注（Markdown 格式）
```

**数据库**：
- 不新增独立表
- `datasources` 表增加字段：`query_guide_updated_at`（最后更新时间）

### 3.2 API 设计

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/datasources/{id}/query-guide` | 获取查询指南内容 |
| PUT | `/api/datasources/{id}/query-guide` | 更新查询指南（备注） |
| POST | `/api/datasources/{id}/query-guide/upload` | 上传文档 |
| DELETE | `/api/datasources/{id}/query-guide/files/{filename}` | 删除文档 |

### 3.3 智能体工具

新增工具 `explore_query_guide`，与现有 `explore_knowledge` 逻辑相似：

```python
@tool
async def explore_query_guide(command: str) -> str:
    """浏览查询指南文档。

    查询指南包含该数据源的业务规则、数据字典、SQL参考等。
    在查询数据前，建议先查阅查询指南。

    常用命令:
    - ls -la : 查看有哪些文档
    - cat *.md : 阅读所有文档
    - grep "关键词" *.md : 搜索特定内容
    """
```

### 3.4 System Prompt 调整

**删除知识库相关描述**，新增查询指南描述：

```markdown
## 查询指南

每个数据源可能有查询指南文档，包含表说明、业务规则、SQL参考等。
在查询数据前，建议先查阅查询指南：`explore_query_guide("ls -la")`
不确定业务逻辑时，务必查阅查询指南。
```

## 4. 前端设计

### 4.1 入口

数据源列表页，每行增加"编辑"按钮，点击弹出侧边抽屉。

### 4.2 抽屉布局

```
┌─────────────────────────────────────┐
│ 编辑数据源                      [×] │
├─────────────────────────────────────┤
│ 基本信息                             │
│ ┌─────────────────────────────────┐ │
│ │ 名称: [生产数据库        ]      │ │
│ │ 类型: MySQL                     │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 查询指南                             │
│ ┌─────────────────────────────────┐ │
│ │ 📄 已上传文档                   │ │
│ │  • 数据字典.md      [删除]      │ │
│ │ [+ 上传文档]                    │ │
│ ├─────────────────────────────────┤ │
│ │ 📝 备注说明                     │ │
│ │ ┌─────────────────────────────┐ │ │
│ │ │ (Markdown 编辑器)           │ │ │
│ │ └─────────────────────────────┘ │ │
│ └─────────────────────────────────┘ │
│                                     │
│           [取消]  [保存]            │
└─────────────────────────────────────┘
```

### 4.3 提示信息

```
💡 查询指南帮助智能体更准确理解数据。内容越规范、越详细，查询效果越好。
   建议包含：表/字段说明、常用SQL参考、业务规则、注意事项等。
   ⚠️ 文档越多，查询响应可能稍慢。
```

---

## 5. 知识库处理

- 保持现有代码不变
- System Prompt 中删除知识库相关描述
- 智能体不再主动提及知识库（无感知）

## 6. 实现范围

1. 后端：新增查询指南服务、API 接口、智能体工具
2. 前端：数据源列表增加编辑按钮、侧边抽屉组件
3. 智能体：调整 System Prompt，新增 `explore_query_guide` 工具
4. 数据库：`datasources` 表增加 `query_guide_updated_at` 字段
