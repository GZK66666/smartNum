# SmartNum 更新日志

所有重要的功能变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [3.5] - 2026-03-21

### 新增

- **RAGFLOW 知识库集成**
  - 集成 RAGFLOW 0.19.1 作为全局知识库
  - 支持企业文档、规章制度、政策法规、公文文件等语义检索
  - 支持文件格式：TXT, MD, DOCX, PDF
  - AI 通过 `search_ragflow` 工具自主检索知识库

### 变更

- **架构调整**
  - 移除本地知识库模块 (knowledge_service.py, knowledge.py)
  - 移除查询指南模块 (query_guide_service.py)
  - 新架构：数据库 (结构化数据) + RAGFLOW(非结构化数据)
  - 智能体自由组合调用工具，自主规划查询策略

### 移除

- 移除 `knowledge_files` 数据库表
- 移除 `datasources.query_guide_updated_at` 字段
- 移除本地知识库文件存储目录

### 修复

- 修复 RAGFLOW API 调用格式 (httpx 不支持 json 参数)
- 修复前端文件列表数据格式映射问题
- 修复相似度显示为 0% 的问题 (兼容 similarity/score 字段)

## [3.4] - 2026-03-17

### 新增

- **查询指南模块**
  - 为每个数据源绑定独立的查询指南
  - 支持上传文档（txt/md/docx/pdf）和手动备注
  - 智能体查询前可自由探索查询指南
  - 侧边抽屉编辑界面

### 变更

- 知识库模块不再在智能体中主动提及（保持代码，无感知）
- System Prompt 调整为查询指南描述

---

## [3.3] - 2026-03-16

### 新增

- **文件数据源支持**
  - 支持上传 CSV、Excel (.xlsx/.xls) 文件作为数据源
  - 使用 DuckDB 进行 SQL 查询，保持与数据库查询一致的体验
  - 前端添加表格格式要求提示，包含示例数据
  - 简化上传流程，选择文件后可直接创建

### 修复

- 修复饼图渲染时分类名称为 null 的问题，自动推断分类字段

### 优化

- 删除不必要的文件转换逻辑，直接使用原始文件查询
- 数据源 ID 生成逻辑统一，确保文件路径与数据库记录一致

---

## [3.2] - 2026-03-15

### 新增

- **知识库系统**
  - 支持上传业务知识文件（TXT、Markdown、Word、PDF）
  - 知识库按数据源隔离，每个数据源独立管理知识文件
  - 智能体可通过 `explore_knowledge` 工具自由探索知识库
  - 支持 grep/cat/ls 等 shell 命令浏览知识内容
  - 文件自动转换为纯文本存储，保留原始文件名便于识别

- **DeepAgents 生产环境重构**
  - 配置 LangGraph checkpointer 实现状态持久化
  - 使用 session_id 作为 thread_id，支持多轮对话状态保持
  - 导出文件持久化到数据库，支持过期自动清理
  - 使用原生异步流式 API，移除线程与协程混合

- **知识库管理界面**
  - 拖拽上传文件
  - 文件列表展示（文件名、类型、大小、上传时间）
  - 文件预览模态框
  - 深色主题科技感风格

### 优化

- 工具定义简化，知识库工具统一为 `explore_knowledge`
- 系统提示词精简，让智能体自主判断何时查阅知识库

---

## [3.1] - 2026-03-15

### 新增

- **智能体执行步骤展示优化**
  - 移除工具名称映射，直接显示原始工具名（list_tables、get_table_schema、run_sql 等）
  - 每个步骤显示类型标签（tool_call/tool_result）
  - 合并 tool_call 和 tool_result 为一个步骤，同时展示输入输出
  - 流式结束后自动收起执行步骤面板

- **Markdown 渲染优化**
  - 添加完整的 Markdown 样式支持（表格、代码、列表、引用等）
  - 后端强化系统提示词，要求使用 Markdown 表格格式输出

- **仪表盘优化**
  - 会话数显示真实总数，不受分页限制
  - 使用独立的 queryKey 避免缓存冲突

### 修复

- 修复历史对话加载时智能体执行步骤显示不正确的问题
- 修复步骤详情无法展开的问题
- 修复可视化请求后仍显示文字内容的问题
- 修复导出请求只显示文字信息的问题
