# SmartNum 更新日志

所有重要的功能变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [3.1] - 2026-03-15

### 新增

- **智能体执行步骤展示优化**
  - 移除工具名称映射，直接显示原始工具名（list_tables、get_table_schema、run_sql 等）
  - 每个步骤显示类型标签（tool_call/tool_result）
  - 合并 tool_call 和 tool_result 为一个步骤，同时展示输入输出
  - 流式结束后自动收起执行步骤面板

- **Markdown 渲染优化**
  - 添加完整的 Markdown 样式支持（表格、代码、列表、引用等）
  - 后端强化系统提示词，要求使用 Markdown 表格格式输出

- **仪表盘优化**
  - 会话数显示真实总数，不受分页限制
  - 使用独立的 queryKey 避免缓存冲突

### 修复

- 修复历史对话加载时智能体执行步骤显示不正确的问题
- 修复步骤详情无法展开的问题
- 修复可视化请求后仍显示文字内容的问题
- 修复导出请求只显示文字信息的问题

## [3.0] - 2026-03-10

### 新增

- **用户认证系统**
  - 用户注册功能
  - 用户登录功能
  - JWT Token 认证
  - 密码 bcrypt 加密

- **数据持久化**
  - MySQL 持久化存储
  - 用户、数据源、会话、消息全部持久化
  - 数据库初始化脚本

- **数据导出功能**
  - CSV 格式导出
  - Excel (.xlsx) 格式导出
  - 下载卡片组件

### 变更

- 重构前端，使用 React Query 进行数据管理
- 移除 Zustand 状态管理，简化架构
